# HR Analytics Dashboard (Microsoft Excel)

## 📌 Project Overview

This project is an interactive HR Analytics Dashboard built using Microsoft Excel.

The dashboard helps HR managers analyze employee performance, salary, promotions, overtime, and workforce distribution using Pivot Tables, Pivot Charts, KPI Cards, and Slicers.

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- KPI Cards
- Slicers
- Data Cleaning

---

## 📊 Dashboard KPIs

- Total Employees
- Average Salary
- Average Performance Rating
- Average Overtime Hours
- Promoted Employees
- Average Years of Experience

---

## 📈 Dashboard Visualizations

- Employees by Department
- Average Salary by Department
- Average Performance Rating by Department
- Average Overtime Hours by Department
- Gender Distribution
- Work-Life Balance Distribution

---

## 🎯 Features

- Interactive Dashboard
- Dynamic Slicers
- Data Cleaning
- Business Insights
- HR Performance Analysis

---

## 📂 Dataset

- 1000 Employee Records
- 11 HR Attributes
- Cleaned and analyzed using Microsoft Excel
